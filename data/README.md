# Data

This folder contains the data files used for the study “Profiling SDG Involvement in Croatian SMEs: Combining Count Models, Machine Learning and Sustainability-Definition Coding.” 

## Files

| File                                   | Description                                                                                                                                                               |
| -------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `Survey_Results.xlsx`                  | Encoded survey dataset. Each row represents one firm/respondent, and each column represents one coded survey variable. The file contains 103 responses and 196 variables. |
| `feature_map_coded_questionnaire.json` | Feature map/codebook for the encoded questionnaire. It explains how each survey question was transformed into variables used in the Excel dataset.                        |

## Dataset structure

The survey covers firm profile, sustainability practices, management and financing aspects, SDG awareness, SDG relevance, and SDGs directly related to business operations.

The main analytical outcome can be constructed from the variables related to Question 37 (`q37_sdg_vezani_uz_poslovanje`). These variables indicate which UN Sustainable Development Goals are directly connected with each firm's business operations.

## Note

The Excel file contains coded research data, while the JSON file documents the meaning of the coded variables. These two files should be used together.
